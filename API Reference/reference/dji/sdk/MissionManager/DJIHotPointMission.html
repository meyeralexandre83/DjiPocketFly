Class Overview

The class represents a hotpoint mission, which can be executed by the mission manager. In a hot point mission, the aircraft will repeatedly fly circles of a constant radius around a specified point called a Hot Point. The user can control the aircraft to fly around the hotpoint with a specific radius and altitude. During execution, the user can also use the physical remote controller to modify its radius and speed. It is not supported by Mavic Pro when using WiFi connection.

Fields

public double altitude

Sets the altitude of the hot point orbit in meters with range [5,120] meters. The altitude is relative to the ground altitude from which the aircraft took off.

public float angularVelocity

Sets the angular velocity (in degrees/second) of the aircraft as it orbits the hot point. The default value is 20 degrees/second. The angular velocity is relative to the orbit radius. You can use the maxAngularVelocityForRadius(double radius) method to get the maximum supported angular velocity for a given radius.

public DJIHotPointMission.DJIHotPointHeading heading

Aircraft's heading while flying around the hot point. It can be pointed towards or away from the hot point, forwards or backwards along its flight route, or be controlled by the remote controller.

public boolean isClockwise

true if the aircraft is to travel around the hotpoint in a clockwise direction.
