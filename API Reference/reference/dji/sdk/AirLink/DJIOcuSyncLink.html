public void setChannelNumber (Integer midPointIndex, DJICommonCallbacks.DJICompletionCallback callback)

Selects the channel by an index. The OcuSync link allows user to use a channel with the selected bandwidth within 2400.5 MHz to 2482.5 MHz. Indices from 1001 to 1084 are assigned discretely to the full frequency range. 1001 is mapped to 2400.5 MHz and 1084 is mapped to 2483.5 MHz. The frequency mapped by two adjacent indices has 1MHz interval. User can select the channel with the mid-point index. The valid range for the mid-point indices are different with different bandwidth. Use getChannelNumberValidRange to check the valid range. It can be set only when the selection mode is DJIOcuSyncChannelSelectionModeManual.

Parameters
midPointIndex Mid-point index of the selected channel.
callback Completion block that receives the setter result.
