# DjiPocketFLy

## Introduction

Ce projet permet la recuperation des donnnés gps du drone ainsi que le retour fpv

## Pré-requis

 - Android Studio 2.0+
 - Android System 4.1+
 - DJI Android SDK 4.0.1

##Demarrage du drone

- LA TÉLÉCOMMANDE DU DRONE DOIT ÊTRE ALLUMÉ​
- LA TÉLÉCOMMANDE DOIT ÊTRE APPAIRÉ AU DRONE​
- LE DRONE DOIT ÊTRE ALLUMÉ ​
- LE SMARTPHONE DOIT ÊTRE CONNECTER EN WIFI SUR LA TÉLÉCOMMANDE AVEC POUR SSID PHANTOM3​
- AVOIR UNE CONNECTION ACTIVE AVEC LE RESEAU CELLULAIRE (3G OU 4G)


## Lancement de l'application à partir d'android studio

- Recuperer le git
- Lancer DjiPocketFLy
